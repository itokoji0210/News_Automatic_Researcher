## 選定ルール
- 根拠URLのある候補だけを扱う
- 公式情報、報道ソース、自治体/施設/企業発表を優先する
- 70点未満の候補は原則落とす
- 単なるグルメ、物産イベント、PR色のみ、夜間のみ、撮影許可が現実的でないものは除外する
- 写真・テレビ映像で成立する現場を最優先する
- ドローンは、上空から規模や変化が分かる場合だけ高評価にする
- 既報でも、後追いできる現場、別角度、定点比較、翌日以降も追えるものは残してよい
